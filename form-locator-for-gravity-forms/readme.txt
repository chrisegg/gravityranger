Lists WordPress pages and posts that contain Gravity Forms block or shortcode, including those deleted, trashed, and inactive.

== Description ==

Creates an admin page that lists WordPress pages and posts that contain Gravity Forms block, shortcode and login form along with the form status including those deleted, trashed, and inactive.


== Frequently Asked Questions ==

= Does it only work with Gravity Forms =

Yes.


== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Click on the `upload` link to upload `form-locator-for-gravity-forms.zip`
4. Click on `Activate plugin`
